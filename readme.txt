Easily add Copy and Share buttons to blockquotes in WordPress posts, including automatic post URL copying.

== Description ==
This plugin adds Copy and Share buttons to blockquotes in WordPress posts with automatic post URL inclusion.

== Installation ==
1. Upload `quote-status-copy-share-by-adittaw` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==
= How do I add the Copy and Share buttons? =
The buttons are automatically added to all blockquote elements on your posts/pages.
